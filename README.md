### 🏪 Omar Store — متجر مكملات رياضية

> منصة تجارة إلكترونية كاملة من الصفر مع لوحة إدارة محمية، API كاملة للمنتجات، نظام خصومات مرن، ورفع صور — كل ده مربوط بـ Supabase PostgreSQL.

**✅ Features:**
- 🛍️ عرض منتجات ديناميكي مع فلترة بالفئات
- 🔐 Admin Panel محمي بدون Framework
- 🏷️ نظام فئات CRUD كامل محفوظ في Supabase
- 💸 نظام خصومات (مبلغ ثابت / نسبة مئوية)
- 📱 Responsive على كل الأجهزة
- 🖼️ رفع صور مع Preview فوري
- ⭐ إدارة آراء العملاء

### 🏆 STEM Sadk Challenge — منصة التحديات اللغوية

> نظام تفاعلي متكامل لإدارة وحل التحديات اللغوية والمسابقات البرمجية لطلاب **SADK Deutsch Klub** — مصمم لتقديم تجربة مستخدم سلسة مع تقييم فوري ومستويات متعددة.

**✅ Features:**
- 🎯 تحديات مصنفة حسب المستوى (A1 → C2)
- ⚡ نظام تقييم واختبار فوري
- 📊 Leaderboard تفاعلي للمراكز والنقاط
- 🔥 نظام Streak لتتبع الأيام المتتالية
- 🎨 واجهة Responsive متوافقة مع كل الأجهزة

[![Visit STEM Sadk](https://img.shields.io/badge/▶_شوف_المشروع-stem--sadk.web.app-EAB308?style=for-the-badge&logoColor=black)](https://stem-sadk.web.app/challenge.html)
